# kaiz

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

To build:

- Linux:
```bash
$ ~ bun build --compile --minify ./index.ts  --target=bun-linux-x64 --outfile=bin/kaiz
```

- Windows:
```bash
$ ~ bun build --compile --minify ./index.ts  --target=bun-windows-x64 --outfile=bin/kaiz
```
